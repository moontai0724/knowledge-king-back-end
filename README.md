# Description

Knowledge King is questioning and answering system developed for evaluate members in an organization.

# Commands

## Installation
### Install dependencies
```bash
yarn install
```

### Setup environment variables

Copy .env file and set it up.

```bash
cp .env.example .env
```

## Running the app

### development
```bash
yarn run start
```

### watch mode
```bash
yarn run start:dev
```

### production mode
```bash
yarn run start:prod
```

## Test

### unit tests
```bash
yarn run test
```

### e2e tests
```bash
yarn run test:e2e
```

### test coverage
```bash
yarn run test:cov
```
